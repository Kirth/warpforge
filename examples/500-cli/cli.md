CLI
===

## check

### Check a Formula is Valid

[testmark]:# (checkformula/cmd)
```
warpforge check formula.json
```

### Check a Plot is Valid

[testmark]:# (checkplot/cmd)
```
warpforge check plot.json
```

## run

### Execute a Formula

[testmark]:# (runformula/cmd)
```
warpforge run formula.json
```
